Bio

I come from a rural town in North Carolina, USA. I've been fascinated and enamored by computers since I was a kid - from game consoles to mobile phones and PCs. I got into full-stack web development in ~2015. I've been learning since, being sure to keep up with recent technologies and frameworks.
